### Task 2: Implement updateTransaction method

**Goal:** Add transaction update capability to ActualApiService

**Specification:**

```typescript
/**
 * Update a transaction's category
 * @param transactionId - ID of transaction to update
 * @param categoryId - ID of category to assign
 * @returns Updated transaction object
 * @throws Error if transaction or category not found
 */
async updateTransaction(
  transactionId: string,
  categoryId: string
): Promise<Transaction>
```

**Implementation:**
1. Add method to ActualApiService class
2. Call underlying @actual-app/api update method
3. Return updated transaction
4. Error handling:
   - Transaction not found → throw Error("Transaction not found")
   - Category not found → throw Error("Category not found")
   - API errors → pass through with context

**Code location:** `src/services/actual-api.ts`

**Example implementation pattern:**
```typescript
async updateTransaction(
  transactionId: string,
  categoryId: string
): Promise<Transaction> {
  if (!this.initialized) {
    throw new Error('API not initialized');
  }

  // Verify category exists
  const categories = await this.getCategories();
  const category = categories.find(c => c.id === categoryId);
  if (!category) {
    throw new Error(`Category not found: ${categoryId}`);
  }

  // Update transaction (actual API call)
  const updated = await actual.updateTransaction(transactionId, {
    category: categoryId
  });

  if (!updated) {
    throw new Error(`Transaction not found: ${transactionId}`);
  }

  return updated;
}
```

**Success:** Method exists, compiles, error handling in place
